## `files_move.cpp`

搜索并移动文件的 CLI 程序。

Usage:

`files_move -s <string> -l <line> -d <destination>`

Move .txt files which contain `<string>` in line `<line>` to directory `<destination>` in current directory, including sub directories recursively.
